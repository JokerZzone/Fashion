订单状态

您在本网成功提交订单后，可以点击网页顶部“我的帐户”查询您的订单，根据订单状态可以了解订单的处理进度：
1.未确认订单(付款方式为在线支付除外)：即您刚刚提交的订单，并且您的订单还未经过我们的工作人员处理。这时您可以对订
单信息(除商品信息以外)进行修改。
2.等待付款：表示您的订单已经经过我们的工作人员确认；如果选择的是银行或邮局汇款，那么您可以立即办理汇款了。
3.发货中：表示商品已经处于发货及配送环节。
4.交易完成：表示该订单我们已经收到货款并已将商品发出。

查询、修改或取消订单

1. 进入“我的帐户”： 本功能只有N°5时尚广场注册会员可以使用(注册成会员能为您的购物提供很多便利)；已注册会员输入用户名和密码后，点击进入网页顶部“我的帐户”即可查询您的订单情况；在订单处于“未确认订单”状态下，您还可以对您的订单信息(除商品信息以外)进行修改。
2. 通过“在线客服”：您可以进入“在线客服”向客服人员咨询，客服离线情况下可以给客服留言。
3. 通过“E-mail”：您可以通过发送邮件至 来查询订单情况，我们将尽快解答您的疑问。
4.通过“客服热线”： 您可以通过拨打我们的客服热线来查询您的订单情况。

被视为无效的订单

以下订单将会被视为无效订单，我们将会把该订单改为“已取消订单”
1.订单信息不完整而导致我们无法与您取得联系的。
2.选择了“银行汇款”或“邮局汇款”，但超过一个月仍然未收到您的汇款的订单。
3.网上支付没有成功的订单。
4.重复的订单。

网上支付未成功

如果您提交订单时选择了网上支付，那么您必须在订单生成页面点击订单号下方的支付链接，到支付平台进行支付；如果您在支付过程中遇到网络中断、因安装IE浏览器128位加密程序导致系统重启、填写错误、支付卡余额不足等原因而导致网上支付没有成功，那么此订单就不能再进行支付了，您可以：
1.重新下订单进行支付；前边未支付成功的订单我们将会改为“已取消订单”。
2.与客服中心联系将订单更改为其他付款方式。
