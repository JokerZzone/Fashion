网上下单

您可以通过浏览我们的网站 ( http://www.no5.com.cn )，选择需要的商品，然后点击页面上的“放入购物车”按钮，并按网页上的提示，先注册成为N°5时尚广场会员( 点击查看会员优惠政策 )，然后选择您的收货地区、配送方式及付款方式、填写您的正确地址及联系方式，便能轻松完成网上购物。
*订单号产生即表示订单已经生效，选择送货上门的用户我们会与您电话确认，选择汇款的用户即可办理汇款。

电话订购

您可以通过拨打我们的订购电话来订购您喜欢的商品。
客服中心： 400-883-6655　 010-5166 6655
*我们的工作时间：每天早9:00到晚上18:00 ；节假日不休息(特殊情况见网页通知)。

邮寄订购

在您选择好要购买的商品后，可以将您要订购商品的货款加上我们为您送货的费用( 点击查看关于配送)，一起通过邮局汇款给我们。
汇款时请注意：
1.请详细准确填写汇款人地址、姓名及邮编，我们将以此地址给您邮寄商品；
2.请务必在汇款单附言内写明您要的商品名称及数量、选择的邮寄方式、和您的工作时间联系电话(如果因为字数太多无法全部填写，请优先填写电话号码，我们收到款后再于与您电话联系)

传真订购

您可以通过我们的在线客服来订购您喜欢的商品(在线客服离线的情况下可以通过留言订购)。
(点击进入“在线客服”)

1.在线浏览可以让您更及时全面的了解商品的信息及促销活动；
2.注册成会员订购您知道自己的订单号更方便查询；
3.您可以随时登录“我的帐户”查看管理您的帐户信息；
4.您可以享受我们的所有会员优惠活动。
