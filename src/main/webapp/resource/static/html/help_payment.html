在线支付

您选购商品及下单的过程是在本网进行的，当您开始在线支付时，您就已经离开本网，并进入了所选银行的网页，所以您填写银行卡资料时，实际上是在银行的网站进行的；因此，对于您的信用卡信息，除银行外其他人是无法进行收集、查看的，请您不必担心您的信用卡资料会泄露。此外，使用时在线支付时需安装认证证书，信用卡的信息就会以高位数加密的方式进行传输。因而，您完全不必担心您的信用卡信息会被破译。 详细情况请点击这里
